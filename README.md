# NovaAI Landing Page

A modern, responsive landing page for a fictional AI startup called **NovaAI**.

The project was built to practice and demonstrate core frontend development skills, including responsive layouts, semantic HTML, CSS styling, and JavaScript interactivity.

[🚀 Live Demo]((https://novaai-landing-page.netlify.app))



---

## 📖 Overview

NovaAI is a fictional company that provides AI-powered productivity and automation solutions.

This landing page showcases:

* Responsive design
* Modern user interface
* Mobile-friendly navigation
* Interactive elements
* Professional landing page structure

---

## ✨ Features

* Responsive navigation bar
* Hero section with call-to-action buttons
* Features section
* Pricing section
* Customer testimonials
* Contact form interface
* Footer with navigation links
* Mobile navigation menu
* Smooth scrolling
* Hover effects and transitions
